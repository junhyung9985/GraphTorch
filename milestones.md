# GraphTorch Milestones

## ✅ Completed (MVP)

* Graph-based model editor (React Flow)
* Node/Edge creation and editing
* Validation (DAG + rules)
* Shape inference
* PyTorch code generation
* Multi Input / Output support
* Preset loading
* JSON Save / Load (modal)
* Edge-attached collapsible Palette / Results panels
* Floating property inspector
* Frontend (Next.js) + Backend (FastAPI) integration
* Deployment setup (Vercel + Render)
